
# 🎬 Box Office Trends (1980 - 2020) | Power BI Analytics Project

## 📘 Project Description

This Power BI dashboard explores 40 years of global **box-office performance data** to identify trends in movie production, revenue generation, profitability, and key contributors to commercial success. The project demonstrates how Power BI can be used to deliver meaningful insights from large historical datasets, making it valuable for film industry analysts, production houses, data professionals, and dashboard enthusiasts.

---

## 🎯 Objectives

- Analyze how production budgets and profits evolved over time  
- Identify top-performing production companies, directors, and actors  
- Discover which genres and runtime categories yield the highest profits  
- Visualize country-wise revenue performance  
- Present a clean, interactive reporting experience with slicers and filters

---

## 🧠 Key Insights Derived

- **Total Budget Spent (1980–2020):** `$195 Billion`  
- **Total Profit Earned:** `$364.43 Billion`  
- **Overall Profit-to-Budget Ratio:** `1.87x`  
- **Average Gross-to-Budget Ratio:** `6.39x`

- 🎬 **Highest Grossing Movie:** *Avatar* with $2.6B in revenue  
- 🧑‍🎬 **Top Earning Actor:** *Robert Downey Jr.* – $9.56B  
- 🎥 **Leading Director:** *Steven Spielberg* – $7.7B  
- 🌍 **Most Profitable Region:** *North America*

## 📊 Dashboard Features

### 📈 Yearly Trend Analysis

- Line chart showing year-on-year growth of budget, gross, and profit  
- Profitability trend line and ratio visualized across decades  
- Impact of key movie releases visualized via noticeable jumps in charts

### 🏢 Top 10 Production Companies

- Bar chart comparing cumulative earnings of major studios like:
  - Universal Pictures
  - Warner Bros.
  - Twentieth Century Fox
  - Columbia Pictures

### 🎬 Directors & Actors Performance

- Bar charts showing top directors and actors ranked by total box-office collections  
- Includes names like James Cameron, Anthony Russo, Tom Hanks, and Will Smith

### 📍 Country Performance Map

- Interactive world map powered by Bing  
- Size of bubbles represents revenue from each country  
- Highlights top-performing markets like North America, Europe, and Asia

### 🧾 Final Report View

- Tabular format with movie-level granularity:
  - Movie Name
  - Director
  - Genre
  - Score
  - Votes
  - Budget, Gross, Profit
  - Profit to Budget Ratio

### 🌀 Genre & Runtime Analytics

- Donut chart to visualize distribution and performance of genres:
  - Action, Animation, Comedy, Drama, Adventure  
- Bar chart segmenting runtime buckets:
  - 90–120 mins performed best in terms of returns  
  - >150 mins and <90 mins showed lesser commercial appeal

## 📁 Dataset Overview

| Field | Description |
|-------|-------------|
| Year | Release year of the movie |
| Movie Name | Title of the film |
| Director | Movie director |
| Genre | Film genre (Action, Drama, etc.) |
| Score | Average rating |
| Votes | Number of user votes |
| Budget | Production budget (in millions) |
| Gross | Total box office collection |
| Profit | Gross - Budget |
| Country | Origin or major market |
| Runtime | Duration of movie |
| Actor | Leading actors |
